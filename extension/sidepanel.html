Custom Tools Guide

              
            

            

              
How To Add A Tool (Step By Step)

              
    
                
  1. Go to Settings ‚Üí Custom Tools ‚Üí + Add Tool.
    2. 
                
  2. Set a short Name (example: my-nuclei).
    3. 
                
  3. Set Command to the executable (example: nuclei).
    4. 
                
  4. Set Arguments Template with placeholders (example below).
    5. 
                
  5. Save, then open Tools tab and run it with a target URL.
    6. 
              

              
Available Placeholders

              
    
                
  • {target}: full target URL from Tools tab.
    • 
                
  • {domain}: domain extracted from target.
    • 
                
  • {output}: auto-generated output filename.
    • 
                
  • {wordlist}: selected wordlist path.
    • 
                
  • {extra_args}: free text from Extra Arguments field.
    • 
              

              
Safe Examples

              
Nuclei:

              nuclei -u {target} -o {output} {extra_args}
              
Nmap:

              nmap -sV -Pn {domain} -oN {output} {extra_args}
              
Tip: first run with no {extra_args}, then
                add flags progressively.

AI Integration Guide

        
      

      

        
Quick Setup (Step By Step)

        
    
          
  1. Open Settings ‚Üí AI Agent Integration.
    2. 
          
  2. Enable AI Agent and choose a Provider.
    3. 
          
  3. Set the Model and Endpoint if needed.
    4. 
          
  4. Add API key (for cloud providers) or leave empty for local Ollama.
    5. 
          
  5. Choose AI Review Scope and Max AI Reviews / Scan.
    6. 
          
  6. Click Test Connection, then Save AI Settings.
    7. 
        


        
Provider Presets

        
    
          
  • OpenAI (GPT): endpoint default is https://api.openai.com/v1/chat/completions, API key
            required.
    • 
          
  • Anthropic (Claude): endpoint default is https://api.anthropic.com/v1/messages, API key
            required.
    • 
          
  • Google Gemini: use AI Studio API key, endpoint can stay default.
    • 
          
  • Ollama Local: http://127.0.0.1:11434/api/generate with model like
            llama3.1:8b.
          
    • 
          
  • Ollama in LAN/VM: use http://<vm-ip>:11434/api/generate and open firewall/port
            11434.
    • 
          
  • Custom Endpoint: endpoint must accept JSON and return text/JSON model output.
    • 
        


        
How AutoScanner Uses AI

        
    
          
  • Local scanner checks run first. AI is only used for triage and prioritization.
    • 
          
  • AI never replaces deterministic confirmation logic (XSS/SQLi/LFI/Traversal/OAST checks).
    • 
          
  • Use Mark TP/Mark FP in finding details to improve future AI triage.
    • 
          
  • You can run AI Re-check from a finding detail modal.
